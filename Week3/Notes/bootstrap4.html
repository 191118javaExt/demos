Welcome to Bootstrap

Styling all your pages with your own CSS seems like a lot of work, especially when you have to think about responsive design and all the different styles you would have to write just to get somethign to look nice. Isn't there an easier way???

Yeah! It is called Bootstrap! Bootstrap is a responsive, mobile-first CSS framework - it is a bunch of pre-written CSS styles (and optional JavaScript) that you can use. No more developing your own CSS from scratch - just use the styling that Bootstrap provides you by default, mainly by including classes on your HTML elements.

So, how to get started? There are two main ways:

  1. Use thier CDN (Content Delivery Network). Just inlcude a link tag with the right URL
  2. Download Bootstrap yourself and link it locally

Bootstrap is a large framewokr with many ready-to-go components and we won't go through all of them, but the most critical aspects of Bootstrap are

  • The grid system - within a "container" you can create rows, which each have 12 columns
  • Responsive by default - use different classes to collapse columns on smaller devices
  • Components - headings, lists, tables, forms, dropdown menus, modals and more!
  • Utility Classes - use various utility classes to style elements how you would want them
  • Consistency - every elemetn will look and feel unified in its design
