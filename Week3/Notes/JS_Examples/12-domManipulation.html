DOM Manipulation

DOM stands for Document Object Model. It is a representation of all of the elements inside of an html document. You can think of it as a family tree of elements. So, The root element is html. And within the html element, we have its children, head and body. Within body, we have an h1 tag as well as this p tag. Each element contains all of that element's lineage so to speak. There are various JS functions that you can use in order to affect the DOM. This is called DOM Manipulation. There are various ways to get elements:

-id (using the getElementById function)
-name (using the getElementsByName function)
-class (using the getElementsByClassName function)
-etc (And there are many others)
